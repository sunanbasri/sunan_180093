Solusi Numerik Untuk Persamaan Difrensial Biasa¶

Persamaan diferensial biasa adalah persamaan diferensial dimana fungsi yang tidak diketahui (variabel terikat) adalah fungsi dari satu variabel bebas. Dalam bentuk yang paling sederhana, fungsi yang tidak diketahui adalah fungsi real atau fungsi kompleks, tetapi biasanya juga dapat berupa fungsi vektor atau matriks. Selain itu, persamaan diferensial biasa diklasifikasikan menurut turunan tertinggi dari variabel dependen yang muncul dalam persamaan.

Berikut adalah contoh sederhana dari hukum gerak kedua Newton, yang menghasilkan persamaan diferensial. [^1]

Metode numerik untuk persamaan diferensial biasa merupakan metode yang digunakan untuk menyelesaikan pendekatan numerik untuk solusi persamaan diferensial biasa (ODE). Penggunaannya juga dikenal sebagai " integrasi numerik ", meskipun istilah ini juga dapat merujuk pada perhitungan integral .

Ada banyak persamaan diferensial yang tidak bisa diselesaikan dengan menggunakan perhitungan simbolis atau analisis. Namun, untuk tujuan praktis seperti dalam bidang teknik dengan menggunakan pendekatan numerik biasanya sudah cukup. Algoritma yang digunakan di sini untuk menghitung perkiraan seperti itu. Metode yang lain yaitu dengan menggunakan teknik dari kalkulus untuk mendapatkan solusi perluasan deret.[^2]

contoh persamaan diferensial biasa (PDB)

Persamaan DiferensialBiasa Metode Euler Orde 1¶

Suatu PD orde satu dapat dinyatakan secara umum dalam dua bentuk, yaitu:

  1. Bentuk Implisit

​ atau F (x ,y ,y' )= 0...........................(1)

  1. Bentuk Eksplisit

​ atau y ′ = f(x ,y ) . ............................. (2)

Seperti pada matakuliah Kalkulus: y' adalah turunan pertama dari y terhadap variabel x. y'' adalah turunan kedua dari y terhadap variabel x. Dst

Jika diberikan persamaan diferensial biasa orde satu

​ y' = dy/dx = f(x, y); y(x0) = y

Misalkan

​ yr = y(xr)

adalah hampiran nilai y di xr yang dihitung dengan metode Euler. Dalam hal ini

​ xr = x0 + rh, r = 0,1,2,... n.

Metoda Euler diturunkan dengan cara menguraikan y (x r+1) di sekitar x r ke dalam deret Taylor: [^3]

Algoritma PDB Metode Euler¶

- t0, N0 = nilai awal dan titik akhir
- tn=b = batas interval
- ndata = jumlah subinterval

Definisi  fungsi f(x,y(x)) = untuk sisi kanan persamaan diferensial
x:=N0, y:=t0
h:=(tn-t0) / n
for n: = 1 to n
    {y:=y+h f(x,y(x))
    x:= N0+nh
    output: x,y}

Implementasi Code Python PDB Metode Euler Orde 1¶

berikut adalah comtoh penerapan code program python untuk mencari PDB metode Euler Orde 1

import numpy as np
import matplotlib.pyplot as plt

def func(N,t):
    k= 0.02
    return -k*N

N0 = 1000
t0 = 0
ndata = 15
tn = 100
t = np.linspace(t0,tn,ndata)
N = np.zeros(ndata)
N[0] = N0
h = t[1]-t[0]

for i in range(0,ndata-1):
    N[i+1] = N[i]+h*func(N[i],t[i])

k =0.02    
NA = N0*np.exp(-k*t)

plt.plot(t,NA,'red',label='analitik')
plt.plot(t,N,'x',color= 'black', label='Euler')
plt.show()
